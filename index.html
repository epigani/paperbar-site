Install Beta

Download. Drag. Open once.

1

Open the DMG

Drag PaperBar.app from the left onto Applications on the right.

2

Approve the beta

The current beta is not notarized. Control-click PaperBar in Applications and choose Open.

3

If macOS blocks it

Open System Settings > Privacy & Security and choose Open Anyway for PaperBar.

4

Stay current

Use Check for Updates in Settings to download newer beta builds.

This is a one-time Gatekeeper exception for the unsigned beta, not a PaperBar account or cloud permission.

FAQ

Questions researchers actually ask.

Does it work offline?

Most utilities do. DOI lookup, online PDF reference fallback, and update checks need the network because they ask public APIs for metadata.

Does it support BibTeX?

Yes. PaperBar can generate, clean, sort, deduplicate, copy, and export BibTeX.

Do I need a Zotero account?

No. PaperBar is not a reference manager and does not require a Zotero account.

Is my data uploaded anywhere?

Tables, BibTeX files, color tools, word counts, QR generation, and PDF text extraction stay on your Mac. DOI lookup sends a query to Crossref when needed. Online PDF reference fallback can query Crossref and OpenCitations. Update checks query GitHub Releases. There is no PaperBar server.

Does PaperBar need macOS privacy permissions?

Most utilities do not. The unsigned beta may need a one-time approval in System Settings > Privacy & Security. If you use screen color picking, macOS may ask for screen access; PaperBar uses it only to sample the color you select.

Is PaperBar an AI assistant?

No. It is a set of deterministic scientific-writing utilities.

Why does macOS say the developer cannot be verified?

The free beta is not notarized yet. After downloading, open the DMG, drag PaperBar to Applications, then Control-click PaperBar and choose Open. If macOS still blocks it, use Open Anyway in System Settings > Privacy & Security.
